# Financial-Services-Dashboard

  ## 📊 Project Overview
  
This Power BI dashboard provides a comprehensive analysis of financial performance including revenue, profit, transactions, and cancellations. It helps stakeholders make data-driven decisions with interactive visualizations.

## 🎯 Key Insights

- 📉 Revenue decreased by 4.8% compared to last year
- 📈 Expenses increased by 1.2%, impacting overall profitability
- 💰 Profit dropped by 3.8% despite a strong margin of 48.7%
- 📊 Highest revenue generated from Delhi and Marketing category
- 🌍 North region contributes the highest share in revenue and transactions
- ❌ Cancellation rate is higher in specific categories like Software
